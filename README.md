# Cybersecurity Projects

A collection of hands‑on projects demonstrating practical security skills. Each project is designed to be **scoped, repeatable, and measurable** with before/after results.

## Projects
- **Vulnerability Scanning Home Lab (`vuln-lab/`)** — Build a small lab with intentionally vulnerable targets, run discovery and vulnerability scans, **remediate**, and **rescan** to quantify risk reduction.

## How to use this repo
1) Open a project folder and read its `README.md` first.
2) Follow the step‑by‑step instructions.
3) Keep all sensitive data out of the repo. Sanitize any scan outputs before committing.
